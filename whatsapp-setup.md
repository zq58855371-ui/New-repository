# WhatsApp Business API Setup

## Step 1: Get WhatsApp Business API Access
- Go to https://developers.facebook.com/docs/whatsapp/cloud-api/
- Apply for access to the WhatsApp Business Platform
- Set up a Business Account in Meta for Developers

## Step 2: Required Credentials
You'll need:
- Phone number ID
- Access token
- Webhook verification token

## Step 3: Connection Process
Once you have the credentials, we'll configure OpenClaw's WhatsApp channel plugin.

Note: This requires approval from Meta which can take a few days.
